Langkah Setup

Supabase
  1. Run `supabase/001_emumtaz_schema.sql` di Supabase SQL Editor.
  2. Run `supabase/002_seed_minimal.sql` untuk data permulaan.
  3. Import senarai sekolah sebenar ke table `schools`.
  4. Daftar owner, admin sekolah, guru kelas, dan guru subjek.
  5. Sambungkan prototype ini kepada Supabase selepas schema stabil.
